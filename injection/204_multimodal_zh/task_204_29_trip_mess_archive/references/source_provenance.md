# Source Image Provenance

Task 029 source files were regenerated as controlled, noisy document photos.
The trip facts remain synthetic and match `ground_truth.json`; real CC0 imagery
is used only for photographic texture / unrelated-photo realism.

- `Weatherworn top of wooden table.jpg`, Wikimedia Commons, CC0, used as the
  desk/table background for photographed documents.
- `A photo of a cat sitting down.jpg`, Wikimedia Commons, CC0, used for the
  unrelated `doc_03.jpg` distractor.

The generated travel documents intentionally contain synthetic passenger,
booking, ticket, and receipt details to avoid real personal data while making
the benchmark input less OCR-trivial. The source filenames are opaque `doc_XX`
labels and should not be treated as semantic labels.
