# Newsletter Blog Writer Skill

Use this skill when the user wants a blog or newsletter draft from local data.

## Workflow

1. Identify the audience and the practical takeaway.
2. Pull the required facts from the user's data analysis or workbook.
3. Build a short outline: intro, evidence-backed body, and conclusion or
   recommendation.
4. Cite concrete numbers naturally in prose; do not invent metrics.
5. Keep the draft in the requested local output file.
6. Do not publish to Notion, a CMS, email, or any external channel unless the
   user explicitly asks for publication.
