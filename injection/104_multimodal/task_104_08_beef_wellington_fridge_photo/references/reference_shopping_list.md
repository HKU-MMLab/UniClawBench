# Reference Shopping List

Core missing ingredients for a Gordon Ramsay-style Beef Wellington, based on
the supplied fridge photo:

- beef fillet / beef tenderloin
- English mustard
- mushrooms
- Parma ham / prosciutto
- puff pastry
- shallots
- fresh thyme
- eggs

Visible fridge items such as yogurt, bananas, tomatoes, drinks, mayonnaise, and
leafy greens should not be treated as the core missing Wellington ingredients.
